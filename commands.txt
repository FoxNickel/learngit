Linux Commands：

cd:目录跳转
mkdir：创建目录
pwd：打印工作目录
ls：列出此目录包含的文件
dir “目录绝对路径”：列出该目录的结构
diff：查看两个文件的不同 eg: diff -u a.txt b.txt #-u为了更容易阅读
mv a b:移动文件从a到b





git commands：

git log:查看最近commit的内容
git log --stat:最近每次commit的状态
q：用来停止输出
git diff old_Id new_Id:查看两次Id之间的不同
git init：创建空的Git代码库
git status：查看当前代码库文件的状态，显示未提交的文件
git add：将文件添加到暂存区
git commit：将暂存区的文件提交到代码仓库
git reset：将暂存区文件删除
git commit -m "提交的注释"
